# BugBoomer — AI 编译你的测试部世界

> **一句话卖点**：输入"996"，AI 编译出单命+黑盒+高速的世界规则——你不只是在玩炸弹人，你在被 AI 编译的世界里求生。
>
> **30秒演示流程**：输入一句话 → AI编译3条规则 → 世界色调变化 → 产品捣乱/研发写Bug → 放炸弹炸屎山+杀Bug
>
> **AI 挂了也能玩**：无 API Key 时自动启用 5 组应急规则包 + 4 组隐藏组合规则，100% 可演示

## 游戏简介

BugBoomer 是一款 **AI 原生** 的职场主题炸弹人游戏。玩家输入一句话描述项目状态（如"996 福报""产品又改需求了"），AI 实时编译出 3 条游戏规则来驱动整个世界：

- **产品捣乱系统** — 产品经理定期在屎山旁复活代码堆，阻挡你的路
- **研发写 Bug 系统** — 研发定期生成各种 Bug 怪，4 种类型各有独特 AI
- **屎山复活系统** — 被炸掉的屎山会按规则自动愈合

玩家的任务：在 AI 编译的混乱世界中，用炸弹炸死所有 Bug + 清除 70% 屎山 = 通关。

### 四种 Bug 类型

| 类型 | 外观 | 特殊能力 | 击杀方式 |
|------|------|----------|----------|
| 🟢 普通 Bug | 绿色虫子 | 正常移动 | 1 颗炸弹 |
| 👻 幽灵 Bug | 紫色幽灵 | 可穿墙 | 1 颗炸弹 |
| 💙 死机 Bug | 蓝屏方框 | 倒计时爆炸 | 1 颗炸弹（要在它爆炸前消灭） |
| 🔴 P0 Bug | 红色大虫 | 需要炸两次 | 2 颗炸弹 |

### 道具系统

| 道具 | 效果 |
|------|------|
| 🍗 大鸡腿 | 恢复 1 点生命 |
| 🫓 大饼 | 增加 1 颗炸弹上限 |
| 🧋 奶茶 | 增加爆炸范围 |
| ⚡ 加速器 | 10 秒速度翻倍，加速时炸屎山有特殊特效 |

### 特色亮点

- **AI 规则编译器** — 每局输入不同的话，AI 生成不同的规则，无限重玩价值
- **一大波 Bug 预警** — 每 45 秒触发"一大波 Bug 正在奔来"横幅，PvZ 风格颤抖放大动画
- **毒舌吐槽系统** — 80+ 条职场毒舌文案，8 种触发场景，打字机效果
- **弹幕系统** — 80+ 条弹幕覆盖击杀/拾取/死亡全场景
- **加速炸屎山特效** — 加速状态下炸屎山，💩 碎片飞出屏幕 + 绿色冲击波
- **开场动画** — 像素测试员敲门 → 喊"测试部开工了" → 坐到工位 → 屏幕亮起 → 进入游戏
- **CRT 复古滤镜** — FC 像素风 + CRT 扫描线，8-bit 音效

## 玩法

1. **输入一句话** — 如"项目太乱""996""敏捷开发"
2. **AI 编译规则** — AI 将你的描述编译为 3 条游戏规则
3. **世界变化** — 规则改变产品捣乱频率、研发写 Bug 速度、屎山行为等
4. **开始战斗** — WASD/方向键移动，空格放炸弹
5. **目标** — 炸死所有 Bug 怪 + 屎山清除率 ≥ 70%

## 配置 AI 接口（可选）

### 方式一：界面配置（推荐）

1. 打开游戏，在输入界面点击「AI 接口：未配置」展开配置面板
2. 填入 API 地址、API Key、模型名
3. 点击「保存配置」，配置会自动保存到浏览器，下次打开无需重复填写

### 方式二：代码配置

打开 `js/ai/compiler.js`，在文件顶部修改：

```javascript
apiUrl: 'https://api.deepseek.com/v1/chat/completions',
apiKey: 'sk-your-key-here',
model: 'deepseek-chat',
```

**推荐的 API 服务：**

| 服务 | API 地址 | 模型 | 说明 |
|------|----------|------|------|
| DeepSeek | `https://api.deepseek.com/v1/chat/completions` | `deepseek-chat` | 便宜好用，注册送额度 |
| OpenAI | `https://api.openai.com/v1/chat/completions` | `gpt-4o-mini` | 效果好，需海外网络 |
| 智谱 | `https://open.bigmodel.cn/api/paas/v4/chat/completions` | `glm-4-flash` | 国内可用，有免费额度 |

**未配置 API 时**：自动使用离线兜底规则包（5 组预设规则），游戏完全可玩。

### 录制要点

- 用 OBS 录屏，1920×1080，60fps
- 录之前关掉所有通知
- AI 编译环节放慢，让评委看清"输入→规则→世界变化"链路
- 每种输入的色调变化要特写（混沌=暖橙、燃烧=暗红、重构=冷蓝）
- 隐藏规则触发时停顿1秒让评委看到 ✨ 标记
- 录 3-5 次取最好的一条
